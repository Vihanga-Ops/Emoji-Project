# ✨ Emoji Explorer — Advanced Emoji Search Web App

## 🔍 Project Overview

**Emoji Explorer** is a modern, interactive web application that allows users to explore, search, and filter emojis in real-time. The application dynamically fetches emoji data from an external source and presents it through a visually rich and user-friendly interface.

This project focuses on **advanced JavaScript functionality, UI/UX design, and real-time filtering**, making it both functional and engaging.

---

## 🚀 Features

### 🔎 Real-Time Search

* Dynamic search functionality as users type
* Supports:

  * Case-insensitive search
  * Partial matching
* Searches across:

  * Emoji names
  * Aliases
  * Tags

---

### 📂 Category Filtering

* Emojis are grouped into categories
* Interactive category pills allow users to filter results
* Active category highlighting for better UX

---

### 📱 Emoji Grid Display

* Responsive grid layout
* Each emoji card displays:

  * Emoji icon
  * Emoji name
* Smooth animations on load and hover

---

### 📋 Click to Copy Feature

* Clicking an emoji copies it to clipboard
* Toast notification confirms the action

---

### ⏳ Data Fetching

* Fetches emoji data from external source:

```bash
https://akhil-06.github.io/emoji_project/emojiList.js
```

* Custom parsing logic used to handle non-standard JSON format

---

### ⚠️ Fallback Data System

* If API fails, a local emoji dataset is used
* Ensures the app always works (even offline)

---

### 🎨 UI/UX Design

* Dark premium theme 🌙
* Animated gradient blobs background
* Glassmorphism-style components
* Smooth transitions and hover effects
* Sticky search bar for better usability

---

### 📊 Stats & Feedback

* Displays total emojis loaded
* Shows filtered result count
* Empty state UI when no results found

---

### 🧠 Additional Features

* Skeleton loading animation
* Clear search button
* Toast notifications
* Smooth scrolling
* Keyboard-friendly search

---

## 🛠️ Technologies Used

* **HTML5** — Structure
* **CSS3** — Styling, animations, responsive design
* **JavaScript (Vanilla JS)** — Core functionality
* **Fetch API** — Data retrieval

---

## ⚙️ JavaScript Functionalities Used

* Event Listeners (`input`, `click`)
* Fetch API with async/await
* Dynamic rendering of elements
* Filtering arrays based on user input
* Conditional rendering (empty state, fallback)
* Clipboard API (`navigator.clipboard`)
* Error handling using `try...catch`

## ▶️ How to Run the Project

1. Download or clone the repository
2. Open the project folder in VS Code
3. Open `index.html` in your browser
4. Start exploring emojis!

---

## 🎥 How to Use

1. Type in the search bar (e.g., "smile", "heart", "fire")
2. Filter emojis by category using the buttons
3. Click any emoji to copy it
4. Use the clear button to reset search
